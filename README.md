# Petit Loading

Unityのコンパイル中にgifアニメーションを表示するパッケージです。

## 動作環境

- Windows
- Python3
  - Pillowライブラリ

## 使い方

画面上部の`Tools/Petit Loading`から設定ウィンドウが開きます。

ウィンドウでは、有効化、画像フォルダ、表示座標の設定をすることができます。

## インストール用URL (UPM)

```
https://github.com/Naggo/PetitLoading.git?path=src/PetitLoading
```
